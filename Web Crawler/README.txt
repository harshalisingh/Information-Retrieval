**************Instructions to Run the Code*************

Run the webcrawler.py using following command:

1) For unfocused crawling (without keyphrase)
python webcrawler.py seedurl

e.g. python webcrawler.py http://en.wikipedia.org/wiki/Gerard_Salton

2) For Focused crawling (with keyphrase)
python webcrawler.py seedurl keyphrase

e.g. python webcrawler.py http://en.wikipedia.org/wiki/Gerard_Salton "information retrieval"
